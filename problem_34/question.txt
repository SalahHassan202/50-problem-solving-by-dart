Problem 34 — Check Armstrong Number
==============================================================
English:
Write a program that takes a number and checks whether it is an Armstrong number or not.
==============================================================
Arabic:
اكتب برنامج ياخد رقم ويشوف هل هو رقم أرمسترونج ولا لأ.
(رقم أرمسترونج = مجموع كل رقم مرفوع لقوة عدد الأرقام يطلع نفس الرقم)
